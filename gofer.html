Gofer - Send Gifts

Gofer is a smart gifting app that fetches you the right gift.


Visit site

First Interactions.

After the user signs up the user will create the giftee's profile through a chat with an AI assistant.


To save some time of process we made pre-defined answers to tap on, that way the user wouldn't need to type it out the answers

Picking a Gift.

After the profile was created the user will get 3 options to pick from based on the giftee's intersts.


The user can enlarge each gift and get further information as to why that gift is a good fit.

Purchasing.

Once a gift was selected the user will be promped with credit credentials. After the assistant will ask for an address and a date to send the gift.

Adding Friends.

After a giftee profile was created the user will be able to create more profiles that will receive their gifts in a given time.


You can also toggle an "Auto-Shop" & as the name implies it'll pick & send gifts automatically.

Playstore.

Our narrative was that this app can help the giftees as they old by serving them the best gifts with minimum effort.


So we pitched that idea along with a landing page that will link you to Google's Playstore.

Logo Process

The idea for which the 'f' sort of serves you the gift you want stroke a cord with us & decided we should go with that in mind.


To serve our customers the best we can.
